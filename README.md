# Maduna Coaches 

A modern, responsive website for **Maduna Coaches**. A professional transport company based in Matatiele, South Africa, offering taxi and bus hire services nationwide.

> ⚠️ **Work in Progress** — This project is currently frontend only. Backend functionality (booking system, form submission, etc.) is yet to be implemented pending further discussion with the client.

---

## About

This website was built as a client project for Maduna Coaches. It serves as their primary online presence, showcasing their services, fleet, and contact information.

## Pages & Sections

- **Hero** — Full-screen landing section with key value propositions
- **Services** — Scholar Transport, Special Trips, and Local Transport
- **Why Choose Us** — Key selling points with stats
- **Our Fleet** — Toyota Quantum, Mercedes Sprinter, and Luxury Coach
- **Contact** — Contact details and enquiry form
- **Footer** — Quick links and company info

## Tech Stack

- [React](https://react.dev/)
- [Lucide React](https://lucide.dev/) — icons
- [Google Fonts — Montserrat](https://fonts.google.com/specimen/Montserrat)
- Plain CSS

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```
